# PokeDex

## Intruções de como rodar:

### Inicie o banco de dados:
```bash
docker compose up
```

### Inicie o backend:
- Entre no diretório backend:
    ```bash
    cd backend
    ```
- Instale as dependências necessárias:
    ```bash
    pip install -r requirements.txt
    ```
- Inicie o servidor:
    ```bash
    flask run
    ```
